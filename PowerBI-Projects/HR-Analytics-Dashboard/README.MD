# HR Analytics Dashboard (Power BI)

## Project Overview
This Power BI dashboard analyzes employee attrition and workforce trends using HR data. The dashboard provides insights into employee turnover across departments, job roles, gender, age groups, and overtime status.

## Key Metrics
- Total Employees: 998
- Employees Left: 167
- Attrition Rate: 16.73%
- Average Age: 37
- Average Monthly Income: ₹6,620

## Dashboard Features
- Attrition by Department
- Attrition by Job Role
- Attrition by Gender
- Attrition by Age Group
- Attrition by Overtime
- Interactive slicers for Department, Gender, Job Role, and Age Group

## Tools Used
- Power BI
- DAX
- Data Modeling
- Interactive Visualizations

## Key Insights
- Research & Development recorded the highest attrition.
- Employees working overtime showed higher attrition levels.
- Employees aged 20–39 contributed the highest share of attrition.
- Attrition varied significantly across job roles and departments.
