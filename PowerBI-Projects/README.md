# Power BI Projects

This folder contains Power BI dashboard projects developed for data analysis, visualization, and business insights.

Projects:
1. Sales Performance Dashboard
A Power BI dashboard analyzing sales performance across products, regions, and time periods. The dashboard includes key sales metrics, trend analysis, and interactive filters for business decision-making.

2. HR Analytics Dashboard
A Power BI dashboard analyzing employee attrition and workforce trends. The dashboard provides insights into attrition by department, job role, gender, age group, and overtime, along with key HR metrics and interactive slicers.
