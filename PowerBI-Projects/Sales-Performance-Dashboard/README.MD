# Sales Performance Dashboard

## Project Overview
Developed an interactive Sales Performance Dashboard in Power BI to analyze sales trends, product performance, and business KPIs.

## Tools Used
- Power BI
- DAX
- Data Modeling
- Data Visualization

## Key Metrics
- Total Sales
- Total Orders
- Average Sales
- Sales Growth %

## Analysis Performed
- Monthly Sales Trend Analysis
- Product Line Performance
- Country-wise Sales Analysis
- Year-wise Sales Comparison

## Interactive Filters
- Year
- Month
- Product Line
- Country

## Outcome
The dashboard provides interactive insights into sales performance, helping identify trends and support data-driven decision-making.
