# Sales Data Analysis

## Project Overview
Performed sales data analysis using SQL to extract business insights, identify trends, and analyze customer and product performance.

## Tools Used
- SQL
- SQLite

## Analysis Performed
- Total Sales Analysis
- Customer Analysis
- Product Performance Analysis
- Country-wise Sales Analysis
- Monthly Sales Trend Analysis

## Business Questions Answered
- Which country has the most orders?
- Which customers purchased more than once?
- Which product line performs best in each country?
- Are most orders High, Medium, or Low value?
- What are the monthly sales trends?

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- CASE Statements
- Date Functions

## Outcome
The analysis provided insights into sales performance, customer behavior, and business trends using SQL queries.
