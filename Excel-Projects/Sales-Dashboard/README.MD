# Sales Dashboard

## Project Overview
Developed an interactive Sales Dashboard in Microsoft Excel to analyze sales performance, identify trends, and track key business metrics.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

## Key Metrics
- Total Sales
- Total Orders
- Average Sales
- Top Performing Products

## Analysis Performed
- Monthly Sales Trend Analysis
- Product Performance Analysis
- Regional Sales Analysis
- Customer Insights

## Interactive Filters
- Month
- Region
- Product Category

## Outcome
The dashboard provides actionable insights into sales performance and supports data-driven business decision-making.
